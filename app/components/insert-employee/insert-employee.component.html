Formulario de Empleado

El ID debe ser un número válido.
El nombre es obligatorio y debe tener menos de 50 caracteres.
El apellido es obligatorio y debe tener menos de 50 caracteres.
El email debe ser válido.
El número de teléfono debe estar en el formato 123-456-7890.
La fecha de contratación es obligatoria.
El ID de trabajo es obligatorio y debe tener menos de 10 caracteres.
El salario debe ser un número válido.
La comisión debe ser un número válido.
El ID del manager debe ser un número válido.
El ID del departamento debe ser un número válido.
